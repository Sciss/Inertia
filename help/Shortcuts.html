Keyboard Shortcuts

molec tool : d = duplicate selected atoms . escape = abort drag . alt + click = delete atom, shift + click = extend selection, meta + click = send molec to the background (allows to select overlapping molecs). ctrl + shift + click = nudge audio region. ctrl + drag = move / drag atom.

addendum v0.5 : global commands are disabled (for now). they are accessible from each soundfile's waveform window. transport shortcuts are accessible when the waveform display is focussed. new shortcuts: Tab selects the region between adjectant markers. successive Tab pressed cycle forward, Alt+Tab cycles backward, [Alt+]Shift+Tab extends selection. Cursor Up/Down sets timeline position to selection start/stop. Numpad 1/2 accesses rewind and fast forward. Meta+Cursor Right zooms to sample level. M drops a marker (marker view must be enabled). A+Ü/+ nudge volume; P+Ü/+ nudge pitch; T+Ü/+ nudge time. Y split selected molecules along timeline position. F fuse selected molecs. L align selected molec start to timeline position.

Note: the shortcuts have been tested on Mac OS with German Apple Pro keyboard. There may be inconsistencies with other configurations or smaller keyboards (PowerBooks for example). Please report any problems and I try to fix them.

The key modifier Meta is the platform's default menu key, which is the Apple-key on Mac OS and Ctrl on Windows and Linux(?).

Global Commands

Note: windows without menu bars (this help frame for example) do not react to global commands. This will be fixed in a future version.

   Transport:

  • Space or Numpad-0 : toggle play / stop
  • Shift+Space : play at half speed
  • Alt+Shift+Space : play at double speed
  • Numpad-/ : toggle loop playback
  • G : open go-to-time dialog

   Editing:

  • F1 ... Fn : select tool
  • Caps lock : toggle blending mode
  • V : toggle catch mode

Main Frame

  • Return : stop playback and go to start of document
  • Shift+Return : extend selection to document start
  • Alt+Shift+Return : extend selection to document end
  • Cursor Left/Right : scroll to begin/end of selection
  • Cursor Up/Down : set timeline position to selection begin/end
  • Alt+F : fit visible span to selection
  • Alt+A or Meta+Cursor Left : zoom all the way out
  • Meta+Cursor Right : zoom to sample level
  • Delete (Backspace) : remove selected time span
  • Meta+A : select whole document span
  • M : drop marker at current timeline position. only effective if marker display is enabled from the view menu.

Special Gadgets

   Path Fields:

  • Meta+Shift+Numpad-1 ... Numpad-9 : store directory in user preferences 1 ... 9
  • Meta+Numpad-1 ... Numpad-9 : recall directory from user preferences 1 ... 9
